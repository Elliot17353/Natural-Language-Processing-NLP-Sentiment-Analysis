# Natural-Language-Processing-NLP-Sentiment-Analysis
I performed sentiment analysis on text data. I started by cleaning and preprocessing the text data, then used TextBlob to classify each text as Positive, Negative, or Neutral based on sentiment polarity. I then visualized the sentiment distribution using a bar chart and generated a word cloud to identify the most frequent words.

## Project Overview
This project focuses on performing Sentiment Analysis on textual data using Natural Language Processing (NLP) techniques. 
The aim of this project is to analyze text data such as customer reviews or social media comments and classify the sentiment expressed as Positive, Negative, or Neutral.

This project demonstrates the complete NLP workflow including text preprocessing, sentiment analysis, visualization, and word frequency analysis.

## Objectives
The objectives of this project are:
- To preprocess and clean text data
- To perform sentiment analysis using TextBlob
- To classify text into Positive, Negative, or Neutral sentiment
- To visualize sentiment distribution
- To generate a word cloud of frequent words
- To perform word frequency analysis

## Tools and Libraries Used
- Python
- Pandas
- NLTK
- TextBlob
- Matplotlib
- WordCloud
- Jupyter Notebook

## Project Workflow
The following steps were carried out in this project:

1. Import libraries
2. Load the dataset
3. Clean and preprocess text data (lowercase, remove punctuation, remove stopwords)
4. Tokenize text into words
5. Perform sentiment analysis using TextBlob
6. Classify text into Positive, Negative, and Neutral
7. Visualize sentiment distribution using a bar chart
8. Generate a word cloud to show most frequent words
9. Perform word frequency analysis
10. Generate insights from the analysis

## Key Insights
- Most of the text data contained Neutral and Positive sentiments.
- Negative sentiment was lower compared to Positive and Neutral sentiments.
- Frequently used words included emotional and positive words such as joy, life, heart, new, and dreams.
- Sentiment analysis can help businesses understand customer opinions and improve decision-making.

## Conclusion
This project shows how Natural Language Processing can be used to analyze text data and extract meaningful insights. 
Sentiment analysis can help organizations understand customer feedback, monitor brand reputation, and make data-driven decisions.
